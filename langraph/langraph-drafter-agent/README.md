# Drafter Agent

We spend way too much time draftin documents. The plan is:

Create an AI Agentic System that can:
- Speed up drafting documents, emails, etc
- Should have a human-in-the-loop step meaning the human sould be able to provide continuous feedback
- The AI Agent sould stop when the human is happy with the draft
- The System should be fast and able to save the draft in a Google Drive folder.
